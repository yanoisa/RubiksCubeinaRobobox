# RubiksCubeinaRobobox
Willkommen zu Rubiks Cube in a Robobox :D
Der Code umfasst zwei Teile. 
Zum einen der adaptierte "Rubriks Cube Solver". Das Originalprojekt findet man auch unter https://github.com/GottaGoGitHub/RubriksCubeSolver
Um das Programm nutzen zu können, benötigt man tkinter.
Man startet das Programm mit:
'python RunCube.py'

Der andere Teil ist ein Code der von einem Raspberry Pi genutzt wird, um einen Zauberwürfel zu verdrehen und zu nutzen. Hierbei muss ich aber anmerken, dass das Program sehr auf DIE Robobox angepasst ist.
